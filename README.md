# LinkUp - Real-Time Location-Based Chat and Event App

LinkUp is a powerful, interactive Android chat application designed to facilitate seamless communication and enhance social interactions through location-based events. Built using Kotlin and Firebase, LinkUp provides secure OTP-based authentication, individual and group chats, and dynamic, geofenced events that engage users based on their real-time locations.

## 🌟 Features

### 🔐 Secure Authentication
- **OTP-based Authentication:** Ensure secure and quick logins.
- **Firebase Authentication:** Robust backend security provided by Firebase.

### 💬 Real-Time Chat
- **User-to-User Chat:** Search for users and initiate conversations instantly.
- **Group Chats:** Easily create and manage group conversations by selecting multiple users.

### 🎯 Location-Based Events
- **Dynamic Event Creation:** Set up events effortlessly by specifying event name and radius in meters.
- **Automatic Invitations:** Nearby users receive automatic notifications and can easily accept or decline invitations.
- **Radius Tracking:** Participants automatically join or leave events based on their physical location.
- **Invitation Management:** Event creators can resend invitations to users who initially declined or missed the invitation.

### 📝 User Profile Management
- **Profile Customization:** Edit your profile picture and username.
- **Account Management:** Easily log out and manage your account preferences.

### 📈 Comprehensive Event Logs *(Coming Soon)*
- Track entry and exit times of users within event zones.
- Detailed logs enable administrators to monitor user interactions and event attendance effectively.

## 🚀 Use Cases
LinkUp is versatile and can significantly enhance user interaction and engagement in various scenarios, including:
- **Educational Institutions:** Boost participation and interaction in lectures, seminars, and extracurricular activities.
- **Corporate Events:** Simplify networking and participant management at conferences and workshops.
- **Social Gatherings:** Create engaging and interactive experiences at festivals, concerts, and community events.
- **Public Safety:** Monitor and manage crowd density and flow at public events to ensure safety.

## 🛠 Technologies Used
- **Kotlin:** Modern, concise, and safe programming language for Android app development.
- **Firebase Authentication:** Secure and reliable user authentication.
- **Firebase Realtime Database:** Efficient, real-time data synchronization and storage.
